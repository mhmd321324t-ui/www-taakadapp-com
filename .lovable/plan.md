

# خطة التحقق والإصلاح الشامل

## المشاكل المكتشفة

### 1. تحذيرات `forwardRef` في Console
- **`PWAUpdatePrompt`**: يُستخدم كمكون عادي لكنه يُمرر ref ضمنياً — الحل: لفه بـ `forwardRef` أو إزالة المرجع
- **`OccasionBanner`**: نفس المشكلة — يحتاج `forwardRef`

### 2. باقي الصفحات والميزات تعمل بشكل صحيح
بعد مراجعة شاملة للملفات:
- **الصفحة الرئيسية (Index)**: التصميم سليم، البطاقات والأقسام مرتبة
- **أوقات الصلاة**: تعمل مع API Aladhan
- **القرآن**: يعرض السور مع مشغل صوتي
- **الأدعية**: تصنيفات مع بحث
- **التسبيح**: عداد يعمل مع حفظ محلي + Supabase
- **المزيد**: قائمة ميزات + محدد الأذان
- **الإشعارات**: نظام interval كل 15 ثانية + أزرار اختبار
- **الأذان**: محدد الصوت + preview + volume slider
- **NotificationCard**: الرابط يشير لـ `/notifications` ✅

### 3. لا توجد أعطال كبيرة أخرى
النظام يبدو مستقر. المشاكل الوحيدة هي تحذيرات الـ console.
